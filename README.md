# 랜스타 일별 매출 리포트

ERP 판매현황(xlsx)을 업로드하면 자동으로 채널별·상품별 매출, 월간 누적, 일별 추이, 목표 달성율을 시각화하는 정적 웹사이트.

## 배포

Render.com Static Site로 자동 배포됨 (이 저장소에 푸시하면 자동 배포).

## 기능

- 판매현황(ERP 원본) 업로드 → 매출데이터/매출리스트 자동 생성
- 채널별 매출 비교 (쿠팡_로켓, 나비엠알오, 지마켓, 스마트스토어, 오늘의집 등)
- 월간 일별 매출 추이 라인 차트
- 상품별 월 누적 판매 TOP 20
- 월 목표 입력 → 달성율·일진행률·월마감 예상 자동 계산
- 매출데이터.xlsx, 매출리스트.xlsx 다운로드
- localStorage 기반 일자별 데이터 누적 (브라우저에 영구 저장)
